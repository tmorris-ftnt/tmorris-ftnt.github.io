Tims tool shed

  • URL Mask - Convert a list of URL's to FortiGuard format.
  • FortiOS Lookup Tool - Look up available FortiOS firmware by platform.
  • ZTP Tool - Desktop app for ZTP setup in FMG.
  • Fortinet Licence Tool - Desktop app for extracting registration keys from Fortinet licence files.
  • Remove VDOM tool - Python script to remove VDOMs from a FortiGate config file.
  • Yankee Date Fixer - Changes MM/DD/YYYY into the correct DD/MM/YYYY format (and back again).
