FortiOS Breadcrumbs - Visual Studio Code Extension

Displays the current VDOM and place in configuration file.

Install/Setup

Install the Fortigate FortiOS Syntax extension (this will create a FortiGate language type).

Download the foritos-breadcrumb extension here.

Install the extension using the "Install from VISX..." option in the extension pane of vscode.

Notes

Ideally I think breadcrumbs should be implemented into the language extension and use the native breadcrumbs navigation bar that is part of vscode so I haven't uploaded this to the marketplace, it's just here for people who might find this implementation useful.
